# Snake Game 🐍

Hi! 👋  
Welcome to my Snake Game project.

This is a classic Snake Game built using Python, where the player controls a snake to eat food, grow longer, and avoid collisions.

---

## Game-Features

- Smooth snake movement  
- Food appears at random positions  
- Snake grows after eating food  
- Score tracking system  
- Collision detection (wall & self)  
- Game over functionality  

---

## 🧠 Concepts Used

- Object-Oriented Programming (OOP)  
- Classes and Objects  
- Loops (game loop)  
- Conditional statements (if/else)  
- Event handling (keyboard input)  
- Collision detection  
- Python modules (turtle / pygame)
